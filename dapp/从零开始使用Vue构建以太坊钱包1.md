### 使用Vue-cli构建一个Vue的webpack项目

我这里就叫做wallet

#### 1：没有安装Node的话，先装一下node，官网下载就好了

<https://nodejs.org/en/>

#### 2：安装vue-cli

npm install -g @vue/cli

#### 3：检查是否安装成功

vue -V

#### 4：创建一个项目

Vue init webpack wallet
